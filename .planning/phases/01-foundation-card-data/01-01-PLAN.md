  <action>
Create fresh Laravel 11 application with Inertia.js server-side rendering and Vue 3:

1. **Install Laravel 11** (if not already present):
   - If existing project is old React app, create new Laravel project in parent directory first, then move contents
   - Or: Delete old src/ and replace with Laravel structure (recommended for same repo)
   - Run: `composer create-project laravel/laravel temp-laravel` then move contents to current directory (preserving .git and .planning)

2. **Install Laravel Breeze with Inertia + Vue**:
   ```bash
   composer require laravel/breeze --dev
   php artisan breeze:install vue
   npm install
   ```

3. **Verify Vite configuration** (vite.config.js):
   - Ensure inertia plugin is configured
   - Ensure @ alias points to resources/js
   - Template: https://laravel.com/docs/11.x/vite#configuration

4. **Verify Tailwind configuration** (tailwind.config.js):
   - Content array must include: ['./resources/views/**/*.blade.php', './resources/js/**/*.vue']
   - This prevents Tailwind classes not working (Pitfall 4 from research)

5. **Verify app.css** has Tailwind directives:
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```

6. **Verify app.js** sets up Inertia with Vue 3:
   - Uses createInertiaApp from '@inertiajs/vue3'
   - Resolves page components from ./Pages/
   - Mounts to #app element

7. **Keep default Welcome.vue** page to verify setup works

8. **Verify routes/web.php** has Inertia route:
   ```php
   Route::get('/', function () {
       return Inertia::render('Welcome');
   })->name('welcome');
   ```

DO NOT:
- Use Mix (Webpack) instead of Vite
- Skip Tailwind configuration
- Use Vue 2 Options API (use Composition API from default Breeze install)

RESEARCH NOTE: Laravel 11 + Inertia.js + Vue 3 + Vite is the standard stack per RESEARCH.md.
  </action>
